# Sales Dashboard

A React + Vite based dashboard that loads a CSV file, cleans the data, and visualizes it using interactive charts.

## Tech Stack
- React
- Vite
- Bootstrap
- PapaParse
- Recharts

## Features
- CSV parsing
- Data cleaning
- Bar, Line and Pie charts
- Filters (Category, Region, Date)
- Responsive design
- Loading and Error states
- URL filter persistence

## Run Locally

```bash
npm install
npm run dev

Open:
http://localhost:5173
